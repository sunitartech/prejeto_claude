# prejeto_claude

> Projeto de teste — repositório criado para aprender e praticar Git + GitHub.

## 📋 Sobre

Este repositório é um espaço de experimentação com o **Git**, **GitHub** e o **Claude Code**. A ideia é servir como ponto de partida para evoluir para um projeto maior, com versionamento bem feito.

## 🚀 Começando

Clone o repositório na sua máquina:

```bash
git clone https://github.com/sunitartech/prejeto_claude.git
cd prejeto_claude
```

## 🛠️ Tecnologias

- **Git** — controle de versão
- **GitHub** — hospedagem do repositório remoto
- **VS Code** — editor padrão configurado no Git
